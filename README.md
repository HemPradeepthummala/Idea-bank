# STEP-Idea-bank

A shared repository where interns can propose, explore, and evolve project ideas.
Each idea is documented with ownership, motivation, approach.

---

## How to add an idea

1. Run `deno task start`.
2. Give the details asked. `name, idea-title, employee id`.
3. Navigate to the idea file.
4. share your idea

## How to Add an Idea manually
1. Copy `data/idea-template.md`
2. Rename it to `ideas/idea-empployee-id.md`
3. Fill in all sections
4. Add your idea entry to the table below
5. commit and share your idea.
